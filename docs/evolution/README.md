# Co-evolução (consumidor)

Este repo é **CONSUMIDOR** do Onion (porta pública `onion-standalone`). O protocolo canônico
(3 fluxos) vive no core (`onion-evolve/docs/evolution/`). Canais: `inbox/` para sinalizar o core
(upstream) e `inbound/` para receber relatórios de update/anúncios do core (downstream).
